# vue实例

在vue中,数据跟视图是响应式的,即修改数据的同时,视图层也会发生改变.但值得注意的是,只有实例创建的时候,存在的属性,才是响应式的.

如果你想要在晚些时候使用到其他的一些数据,你可以先把这些数据先声明,并且赋予初始值,到需要的时候再进行数据的赋值.

- 这里唯一例外的是,Object.freeze(),冻结属性,这会阻止修改属性.
